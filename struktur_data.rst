Struktur Data di Python
=======================

Python memiliki struktur data bawaan yang sangat berguna.

List (Daftar)
-------------
```
buah = ["apel", "jeruk", "pisang"]
print(buah[0])  # apel
```

Tuple
-----
```
warna = ("merah", "hijau", "biru")
```

Dictionary
----------
```
siswa = {"nama": "Budi", "umur": 17}
print(siswa["nama"])
```

Set
---
```
angka = {1, 2, 3, 3, 2}
print(angka)  # hasilnya {1, 2, 3}
```
